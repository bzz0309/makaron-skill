# Street Paparazzi — 街头爆闪

上传照片 → 街头狗仔闪灯风暴，14 秒 Y2K 粗粝爆闪视频。不躲闪、不惊吓——**主角就是全场焦点。**

## Core Strategy

**纯酷版 4 镜**：Pap Walk 进场 → 闪灯风暴 360° → 面部特写推进 → Icon Freeze 定格。
跳过"惊吓/试探"阶段——主角早已习惯镜头，每一帧都是掌控。

## Identity Anchor（来自用户照片）

用户上传的照片 = 身份唯一来源：
- 面部特征、发型、发色、肤色、性别全部忠于原照
- 穿搭保留用户原照中的服装
- 背景从原照中提取（街道/建筑/城市元素），融入城市夜街场景

## Shot Sequence (14s total, 9:16 vertical)

```
Shot 1 (3s): Pap Walk 进场。主角从城市夜街纵深走向镜头，自信步伐。左右两侧密集直闪灯此起彼伏爆闪，狗仔剪影在边缘模糊晃动。表情冷静、眼神直视不闪躲。重度胶片颗粒。

Shot 2 (4s): 闪灯风暴。镜头围绕主角 360° 弧形移动，闪灯群从四面八方密集爆闪——左、右、前、后同步。主角在闪光中微侧头、嘴角微扬，享受这场风暴。多重闪光残影叠加，色彩分离/色散。

Shot 3 (3s): 面部特写。镜头从全景急速推进到面部特写，闪灯频率达到最高潮。主角直视镜头，下巴微抬，自信微笑。皮肤纹理/雀斑/发丝在爆闪中清晰可见。

Shot 4 (4s): Icon Freeze 定格。最后一道最亮的闪光爆开，主角停步中心，半回眸看向镜头。眼睛直视，似笑非笑。全场闪灯同时达到极致的闪光白——定格在这个画面上。
```

## Style

Y2K direct flash photography, heavy film grain, color dispersion, gritty texture.
**Palette**: carbon black, flash white, cool silver, concrete grey, occasional neon accent bokeh.
Cold, raw, real. PHOTOREALISTIC live-action. NO anime, NO cartoon, NO CGI.
NO warm tones, NO golden, NO beauty filter, NO glow, NO date stamp, NO text overlay.

## Rules

1. 用户照片作为身份锚点，面部特征/发型/肤色/性别全部忠于原照
2. 穿搭保留原照服装，不明显时才用黑色街头风补（皮夹克、细肩带、银链）
3. 背景融入原照环境元素，城市夜街化
4. 配饰只随身体自然动，不单独发光/闪烁
5. 无日期戳、无文字覆盖
6. 全片禁止 warm/golden 色调——Y2K 是冷的、粗的、真的
7. PHOTOREALISTIC ONLY — 严禁动漫/卡通/CGI
